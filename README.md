# Guide System 

1. Create the txt file or folder.
2. Make sure that the file and folder contain text or document.
3. Run the system, select file or folder then upload it.
4. Set the password. ( Make sure to remember if not then the file or folder can't be use again. )
5. Then click the encrypt button.
6. Check the file or folder that it is totally encrypt.
7. For decrypt, upload the file or foler then enter the password for security checking.
8. If the password valid then able to click the decrypt button and will see the pop up message that stated ( 1 file decrypt ), if not then file or folder never will be decrypt.
9. Extra --> If you try to decrypt the file or folder second times it will pop up ( 0 file encrypt ) means the program detect unsual activity. So it will not work due to the file or folder is already encrypt.
